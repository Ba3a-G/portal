# Data API implementation

<MarkdownChipRow labels={["Intermediate", "Tutorial" ]} />


The following data can be fetched from the data API endpoints for the ICRC Rosetta implementation:

- [Network](/docs/rosetta-defi/rosetta/icrc_rosetta/data_api/network.mdx): Fetch network specific information, such as the most recent block and what network Rosetta is connected to.

- [Balances](/docs/rosetta-defi/rosetta/icrc_rosetta/data_api/balances.mdx): Fetch the balance of an `account_identifier`.

- [Blocks](/docs/rosetta-defi/rosetta/icrc_rosetta/data_api/blocks.mdx): Fetch blocks from the ICRC-1 ledger.

- [Transactions](/docs/rosetta-defi/rosetta/icrc_rosetta/data_api/transactions.mdx): Fetch transactions from the ICRC-1 ledger.
